# Nxt Watch App

**Live Demo:** [https://arthamnxtrendz.ccbp.tech](https://arthamnxtwatch.ccbp.tech/)  

---

## Project Overview

**Nxt Watch** is a **YouTube-like video streaming platform** built with **React**. Users can browse, search, and interact with videos across trending and gaming categories. It supports saving favorites, Light/Dark theme switching, and fully responsive design.

---

## Key Features

- **User Authentication:** Secure login with JWT-based session handling.  
- **Home & Trending Pages:** Dynamic display of videos using REST APIs.  
- **Video Interaction:** Like, dislike, and save videos for personalized viewing.  
- **Saved Videos:** View and manage saved videos.  
- **Search Functionality:** Instant search results for quick video discovery.  
- **Dynamic Routing:** Protected routes for Home, Trending, Gaming, Saved Videos, and Video Details.  
- **Responsive Design:** Optimized for mobile, tablet, and desktop.  
- **Theme Switching:** Toggle between Light and Dark themes.  
- **Error Handling:** Custom pages for API failures, empty search results, and 404 Not Found.

---

## Technologies Used

- **Frontend:** React.js, React Router, Styled Components  
- **State Management:** Context API  
- **API Integration:** REST APIs  
- **Libraries & Tools:** react-player, date-fns

## Setup Instructions

1. **Clone the repository:**  
```bash
git clone https://github.com/arthamajay/NxtWatchApp.git
```
2 . **Install dependencies**:
```bash
npm install
```
3. **Start the application**:
```bash
npm start
```
**Open http://localhost:3000**
